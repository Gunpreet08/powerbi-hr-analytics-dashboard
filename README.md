# HR Analytics Dashboard – Power BI

## Project Overview
This project presents an interactive HR Analytics Dashboard developed using Power BI to analyze workforce distribution and employee attrition patterns. The dashboard helps organizations understand employee demographics, attrition trends and department-level workforce insights.

## Key Features
- KPI indicators for Total Employees, Attrition Count, Attrition Rate and Average Salary
- Workforce distribution by department and job role
- Employee attrition analysis across age groups and tenure
- Salary insights and employee demographics
- Interactive filters for department, gender and education level

## Tools Used
- Power BI
- CSV / Excel dataset
- Data visualization techniques
